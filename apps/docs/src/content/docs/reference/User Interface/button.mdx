**Props**

| Name | Type | Description |
| ----------- | ----------- | ----------- |
| `variant` | `'default' \| 'destructive' \| 'outline' \| 'secondary' \| 'ghost' \| 'link'` | The button's variant. Default to `default` |
| `size` | `'default' \| 'sm' \| 'lg' \| 'icon' \| 'icon-sm'` | The button's size. Default to `default` |
