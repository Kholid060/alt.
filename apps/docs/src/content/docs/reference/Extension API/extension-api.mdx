---
title: Extension APIs
slug: reference/extension-api
---

These extension APIs can be used anywhere inside the extension command except for the command with the `script` type. You need to add [permission](http://localhost:4321/extensions/manifest/#manifest-properties) in the extension manifest to use some of the APIs.

export const APIs = [
  'browser',
  'childProcess',
  'clipboard',
  'command',
  'fs',
  'notification',
  'oauth',
  'runtime',
  'shell',
  'sqlite',
  'storage',
  'ui'
];

Available APIs:

</ul>
